ABOUT

  • Hi! I'm Vincent, the dojo master. I've build this search UI for my Coveo Technical test

FUNCTIONALITIES

  • Predefined tabs to help you structure your search
  • Lots of filters for quick and precise results
  • Use complex search criteria with sliders
  • Preview your pokemons in the search results or in the quick suggestion dropdown
  • Awesome 404 page !
  • Custom theme for better pokemon experience

Feedback

  • Webscraping is quite easy with the documentation and the Cov√©o platform

  • Major difficulties with the indexed content. Unable to find a Pokemon if I enter the full name. Always need to use a refined search (@pokname='geodude'). Found that some of my crawlers gave an "NaN" value in the index
  • Sometimes, on pokemondb.net, Pokemons have multiple data sets for the same ID (ex: two descriptions and two images for the same pokemon). It gave me some irregularities in the data.
  • Found the HTML editor WAY too late
  • I'm new to GitHub and Heroku, the deploiement has been .. salty!
