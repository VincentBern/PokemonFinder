ABOUT

  • Hi! I'm Vincent, the dojo master. I've build this search UI for my Coveo Technical test
  • The challenge was to create a search page propulsed by Cloud Platform using Coveo’s recently open sourced JavaScript Search Framework.

FUNCTIONALITIES

  • Predefined tabs with different search criteara
  • Filters for quick and precise results
  • Sliders for complex search criteria
  • Custom templates to peak in every pokemon stats
  • Search as you type
  • "Did you mean ?" function
  • SuggestionBox
  • Awesome 404 page !
  • Custom theme for better pokemon experience

Feedback

  • The data was pretty fun to work with, I mean, who doesnt love those little beasts
  • Webscraping is quite easy with the documentation and the Covéo platform
  • Everything was easy to install locally, even if I didn't knew most of those technologies
  • New pokemons were added during the week, going from 720 to 802 !!
  • Extracting and transforming the childs nodes text using XPath was a pain at first.
  • Major difficulties with my index (NaN errors due to bad scrapping)
  • Sometimes, on pokemondb.net, Pokemons have multiple data sets for the same ID (ex: two descriptions and two images for the same pokemon). It gave me some irregularities in the data.
  • Found the HTML editor WAY too late
  • I'm new to GitHub and Heroku, the deploiement has been .. salty!
