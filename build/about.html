ABOUT

Hi! I'm Vincent, the dojo master.

I've done this search UI for my Coveo Technical test.

FUNCTIONALITIES

Basic filters with using CoveoFacet (species, type, generation)
Advanced filters with using CoveoFacetSlider (height, weight, number)
Autosuggest, Did you mean and result count
Toggling the filters and saving the user choice
Quickstats on every pokemon for better search
Awesome 404 page !

Feedback

  • The theme was pretty fun to work with, I mean, who doesnt love those little beasts
  • Webscraping is quite easy with the documentation and the Covéo platform
  • It was easy to install locally, even if I didn't knew most of those technologies
  • Extracting and transforming the childs nodes text using XPath was a pain at first.
  • Sometimes, on pokemondb.net, Pokemons have multiple data sets for the same ID (ex: two descriptions and two images for the same pokemon). It gave me some irregularities between the slider filters and the quickstats information.
  • Basic html snippets would had accelerate my process. I had hard times figuring out how to implemant OmniBox for exemple.
