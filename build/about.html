ABOUT

  • Hi! I'm Vincent, the Dojo Master. I've build this search UI for my Coveo Technical test
  • The challenge was to create a search page propulsed by Cloud Platform using Coveo’s recently open sourced JavaScript Search Framework.
  • The main goal was to index pokemondb.net and customize the search page to include certain features.

FUNCTIONALITIES

  • Predefined tabs to explore differents caracteristics
  • 4 filters for quick and precise results
  • 7 cursors to perform complex search with many criteria
  • Custom pokemon detail template to peak in every Pokemon's stats
  • Search as you type and "Did you mean ?" function
  • SuggestionBox with small thumbnails
  • Awesome 404 page !
  • Custom 8-pit purple theme for better pokemon experience
  • Mobile enabled

FEEDBACK

  • The data was pretty fun to work with, I mean, who doesn't love those little beasts
  • Web scraping is quite easy with the documentation and the Covéo platform
  • Everything was easy to install locally, even if I didn't knew most of those technologies
  • New pokemons were added during the week, going from 720 to 802 !!
  • Extracting and transforming the children node's text using XPath was a pain at first, but CSS was too limited
  • Major difficulties with my index (NaN errors due to bad scrapping, now fixed!)
  • Some Pokemons have multiple data sets for the same ID (ex: Deoxys and his 3 brothers). It gave me some irregularities in the data.
  • Found the HTML editor WAY too late
  • I'm new to GitHub and Heroku, the deployement has been .. salty!
  • Unable to use the sliders on mobile
  • Didn't had the time to force the Wildcard search (enabled it by default)
